    My Courses

  • C++

    C++ is an object-oriented language and is widely used in the development of operating systems, compilers, games, and more. We learnt the basics of programming using C++, including variables, and many more.

    WEB TECHNOLOGY

    Web technology refers to the tools and techniques used to create and manage websites. In this course, We learnt about web development languages such as HTML, CSS, and JavaScript.

    COMMUNICATION SKILLS

    Effective communication is essential in any field, and this has helped us to develop our communication skills.

    DATABASE CONCEPT

    A database is a collection of data that is organized and stored in a computer system. In this course, we learnt about database management systems, including data modeling, database design, and SQL programming.

    DIGITAL ELECTRONICS

    Digital electronics is the study of electronic circuits that use digital signals to process and transmit information.

    ACCOUNTING

    In accounting, we learnt the principles and techniques used to track and analyze financial information.

    COMPUTER HARDWARE

    This course covers the hardware components of a computer system, including CPUs, memory, storage devices, and input/output devices. We also learn about computer architecture and system buses.

    MINI PROJECT

    A mini project is a small project that allows us to apply the knowledge and skills we have learned in other courses. This project may involve programming, database design, web development, or other areas.
